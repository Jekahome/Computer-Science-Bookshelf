## Джастин Танни

[habr 682150](https://habr.com/ru/companies/ruvds/articles/682150/)

* [Braille Dump](https://justine.lol/braille/). Прямая замена hexdump -C, только с применением шрифта Брайля в Юникоде для отображения шестнадцатеричных кодов 0x81...0xff, тем самым улучшая читаемость двоичного кода. Работает на Linux 2.6.16+, Windows Vista+, MacOS XNU 15.6+, FreeBSD 12+, OpenBSD 6.4+, or NetBSD 9.1+.

    ```bash
    $ curl https://justine.lol/braille/bd.com >bd.com
    $ chmod +x bd.com
    $ echo 'hello→world→' | ./bd.com
    00000000  68 65 6c 6c 6f e2 86 92  77 6f 72 6c 64 e2 86 92  │hello⢚⢠⢁world⢚⢠⢁│
    00000010  0a                                                │◙│
    00000011
    ```
* [Printvideo](https://justine.lol/printvideo.html). Утилита командной строки для визуализации видеороликов в текстовом режиме консоли.
